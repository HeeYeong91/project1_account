## :eyes: 1. 프로젝트 소개
1차 실습 프로젝트 Spring boot와 Java를 활용하여 Account 시스템 만드는 프로젝트 <br />

사용자와 계좌의 정보를 저장하고 외부 시스템에서 거래를 요청할 경우 <br />
거래 정보를 받아서 계좌에서 잔액을 거래금액만큼 줄이거나(결제), <br />
거래금액만큼 늘리는(결제 취소) 거래 관리 기능을 제공하는 시스템입니다. <br />
